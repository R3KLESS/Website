Internet Basics

  • HTTP Requests

    • When we open any website page, it sends a HTTP request via data packets, to ask for the website information.

  • Ip Address

    • Everything on the internet has an ip address , it is unique way to identify each website.

    When we want to open a website we have to know where to send our http requests in order to display the website information.

    This is what the ip address is used for. Think of it kinda like a postcode.

  • Servers

    • Most websites information and data are stored on servers.

    When the servers recieve the HTTP request they send back all the data that makes the website

    As the website data is so large it gets broken down into several data packets which are then pieced back together again when they reach the users computer
